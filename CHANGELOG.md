# Changelog

All notable changes to **Smart File Organizer** will be documented in this file.

The format is based on *Keep a Changelog* and this project follows *Semantic Versioning*.

---

## [1.0.0] - 2026-03-07

### Added

* Initial public release of **Smart File Organizer**
* Desktop GUI built with **PySide6**
* Folder selection interface for organizing files
* Automatic file categorization system
* Splash screen support during application startup
* Logging output panel for file organization status
* Windows executable build using **PyInstaller**
* Windows installer created using **Inno Setup**

### Free Version

* Basic file organization functionality
* One-click folder organization
* Lightweight desktop interface
* Manual release via GitHub Releases

### Pro Version

* License key activation system
* Extended features and improvements
* Automated build and release pipeline using **GitHub Actions**
* Cross-platform builds for **Windows and macOS**

### Infrastructure

* Repository structured for **Free and Pro versions**
* CI/CD pipeline setup for automated builds
* GitHub Releases integration for distribution
